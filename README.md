## Installation

1. Install [PyTorch](https://pytorch.org/get-started/locally/)

2. Clone the OpenKE-PyTorch branch:
```bash
git clone https://github.com/aistairc/EvalMRMforKGE.git
cd OpenKE
cd openke
```
3. Compile C++ files
```bash
bash make.sh
```	
4. Quick Start
```bash
python train_transe_kgrc.py
```
